Release Engineering

This project provides automation for the building of each instream product.

To start the build for a particular component use the folowing from the command line.

Pre Eclipse base builder 3.3M5

java -cp ..\org.eclipse.releng.basebuilder\startup.jar org.eclipse.core.launcher.Main -application org.eclipse.ant.core.antRunner -buildfile build.xml -Dcomponent=svn-pde-build

Post Eclipse base builder 3.3M5

java -cp ..\org.eclipse.releng.basebuilder\plugins\org.eclipse.equinox.launcher.jar org.eclipse.core.launcher.Main -application org.eclipse.ant.core.antRunner -buildfile build.xml -Dcomponent=svn-pde-build

ParameterDescriptionExamples
componentSpecifies the component you intend to build. Information about how to build the component is contained the the subdirectory with the components name.-Dcomponent=orgModeller
configDefines the target platform that this build is for.-Dconfig="*,*,*"
buildDirectoryThe directory where the build will take place. The default is a subdirectory of the current directory called "tempBuild".-DbuildDirectory=D:/tempBuild
baseThis is the location of where the build enviroment will go. Default value is a subdirectory of the current directory call "base".-Dbase=D:/tempEnv
buildTypeSpecifies the type of build.
