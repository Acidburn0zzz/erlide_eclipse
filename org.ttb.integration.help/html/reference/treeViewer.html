Tree viewer View

Using this view you can browse trace results in a form of tree. Each event type is represented by different icon. Tree nodes can be unfolded to get more details about particular event. Every tracing is represented as different tree. Root of such tree displays dates of first and last event.

Tree viewer
When you double click on module node file containing its definition will be opened in editor. The same is with node representing function that was called - file containing its module will be opened and cursor will be set at the begining of function's definition.
