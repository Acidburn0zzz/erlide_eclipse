Main panel

Selecting node

Using list you can select which node will be activated for tracing. This displays nodes started by erlide. You may notice node named "tracing" - this is node on which tracing tool (ttb) runs. You will also see node with a strange name - it is a node which runs in the background of erlide. To update list of nodes (e.g. after starting new node) click Refresh button.

Starting and stopping tracing

To start (stop) tracing click button in the top right corner of the Erlang tracing control panel view, near minimize button.
