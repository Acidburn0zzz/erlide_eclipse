Editor view

The Erlang editor provides specialized features for editing Erlang related files.

Associated with the editor is a Erlang-specific outline view, which shows the structure of the active .erl or .hrl file. It is updated as you edit these files.

The editor includes the following features:

  • Syntax highlighting

  • Content/code assist

The most common way to invoke the Erlang editor is to open a file from the Erlang Navigator.
