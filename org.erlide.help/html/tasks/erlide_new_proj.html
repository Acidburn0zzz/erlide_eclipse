Creating a project

To create a project:

  1. Click File > New > Other... > Erlang Project or Right click in Erlang Navigator > New Erlang Project.


  2. In the New Erlang Project wizard, enter Project Name and chose location


  3. Click Next.
  4. In the Project Build Preferences boxes, type Output, Source and Include folders or use the default values.



  5. To create your project, click Finish.
