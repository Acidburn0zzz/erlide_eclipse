Installation guide


  • Install Erlang R12 or later, if it isn't already present on your system. On Windows systems, use a path with no spaces in it.


  • Install Eclipse. The recommended Eclipse version is the latest 3.3 service pack.


  • If your network uses a proxy, fill in the appropriate data in Window->Preferences->Install/Update->Proxy settings.


  • Download Erlide by going to Window->Software Updates->Find and Install...->Search for new features to install. In the dialog, choose New remote site and enter Erlide as name and http://erlide.sourceforge.net/update as URL. Follow the dialogs with the obvious choices and in the end you will be asked to restart.


  • Restart. Open Window->Preferences->Erlang->Runtime (ERTS) and enter the location of the Erlang installation. A new restart is needed, sorry about that (will fix later).


  • Done! You're ready to start exploring erlide
